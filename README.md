# Mini Product Dashboard – داشبۆردی بەڕێوەبردنی کاڵاکان

ئەم پڕۆژەیە داشبۆردێکی سادە، تامەزرۆز و کاریگەرە بۆ بەڕێوەبردنی کاڵاکان. وەک پانێلی بەڕێوەبردن کار دەکات کە لە یەک شوێنەوە دەتوانیت زیادکردن، دەستکاریکردن، گەڕان، فلتەرکردن و بینینی ستاتەکانی کاڵاکانت بکەیت.  
دیزاین بە شێوەیەک کراوە کە خاوەن سادەیی، سرعت و کاربەر-دووستییە.

---

## خاسیەتەکان

- تامەزرۆز لەسەر مۆبایل، تەبلێت و کۆمپیوتەر  
- زیادکردنی کاڵا  
- دەستکاریکردنی کاڵا  
- گەڕان و فلتەرکردنی کاڵاکان  
- نیشاندانی ستات و داتاکان لە داشبۆرد  
- سیستەمی چوونەژوورەوەی پارێزراو (Authentication)  
- پەڕەی ڕێکخستن و بەڕێوەبردنی هەژماری بەکارهێنەر  
- دیزاینێکی ڕوونی سادە و کاربەر-دووستانە  

---

## تکنۆلۆژیە بەکارهاتووەکان

- **Next.js 14** – فریمۆڕکی پێشکەوتوو بۆ وێب  
- **TypeScript** – بۆ کۆدی پاکتر و هەڵەکەمتر  
- **Tailwind CSS** – بۆ دروستکردنی دیزاینێکی خاوەن جوانی و خێرا  
- **shadcn/ui** و **Radix UI** – بۆ کۆمپۆنێنتی UI ی پڕۆفەشنەل  
- **TanStack Table (React Table)** – بۆ ڕێکخستن و نیشاندانی داتا لە جدۆڵ  
- **React Hook Form + Zod** – بۆ کۆنتڕۆڵکردن و ڕاستکردنەوەی فۆرمەکان
